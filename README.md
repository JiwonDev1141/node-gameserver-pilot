### Reference: Commit Message Rules

- ADD: 새로운 기능 추가
- FIX: 기존 기능 수정
- IMP: 리팩토링 (기능 변경하지 않고 코드나 구조를 수정)
- DOCS: md와 같은 documentation의 변경
- PERF: 성능 개선 작업
- REVERT: 커밋 취소

```
(ex) git commit -m "[ADD] 버튼 추가"
```
